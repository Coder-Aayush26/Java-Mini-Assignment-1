# Java-Mini-Assignment-1
This is the Java Mini Assignment assigned in my university's OOP(object-oriented programming with Java). It was Fun and really involved some good usage of multiple basic OOP concepts, also it used Design patterns to keep the code industry-organized. This is mostly Terminal Based will try to make it into GUI in future.

Database Schema:-
Set your MySQL client username and password in the DBManager.java file in the dao(Data Access Object) package.
Your MySQL client should be open in the background for it to work and transfer the data without any SQL error.
